# Virgo_A
A fictional solar system called `Virgo A` for **Celestia**

- Star: Virgo A
- Planets: 8
  - Class M: 1, Xirius,
    - Moon: 2, xoom, soom

### Installation

Download and extract, or clone entire repository to your Celestia/extras directory and restart Celestia
